## 📖 Описание

Проект представляет собой онлайн-библиотеку аудиокниг с произведениями из школьной программы. Приложение позволяет:

- 📖 Просматривать каталог аудиокниг с обложками
- 🔍 Фильтровать книги по классу, автору и жанру
- 🎧 Переходить к прослушиванию книг на внешних ресурсах
- 📱 Использовать адаптивный дизайн для всех устройств

## 📁 Структура проекта

```
AudioLetovo/

├── books.json          # Данные книг (22 произведения)

```

## 📊 Структура данных

Данные книг хранятся в файле `books.json`. Каждая книга содержит следующие поля:

| Поле       | Тип    | Описание              | Пример                |
| ---------- | ------ | --------------------- | --------------------- |
| `title`    | string | Название произведения | `"Капитанская дочка"` |
| `author`   | string | ФИО автора            | `"Пушкин Александр"`  |
| `class`    | number | Класс программы (7-9) | `8`                   |
| `genre`    | string | Жанр произведения     | `"Роман"`             |
| `coverUrl` | string | URL обложки книги     | `"https://..."`       |
| `audioUrl` | string | Ссылка на аудиокнигу  | `"https://..."`       |

### Пример записи:

```json
{
  "title": "Капитанская дочка",
  "author": "Пушкин Александр",
  "class": 8,
  "genre": "Роман",
  "coverUrl": "https://akniga.org/.../cover.jpg",
  "audioUrl": "https://akniga.org/..."
}
`
```
